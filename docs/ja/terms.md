# Dotbugi 利用規約

::: info
[**Dotbugi**](https://github.com/hs-shell/dotbugi) 利用規約
:::

**施行日:** 2026/05/13

## 1. 概要

本規約は、ユーザーが「HSU Dotbugi」(以下「本拡張機能」)を利用するにあたり必要な条件と責任を定めるものです。本拡張機能をインストール・使用することにより、ユーザーは本規約に同意したものとみなされます。

## 2. サービス内容

本拡張機能は、漢城大学校 LMS(`https://learn.hansung.ac.kr/`)が提供する講義、課題、クイズの情報をユーザーのブラウザ内で統合表示し、任意で Google カレンダーと日程を同期する機能を提供します。

本サービスは**無料**で提供され、漢城大学校または Google と公式な提携関係のない**非公式のサードパーティツール**です。

## 3. ユーザーの責任

ユーザーは以下に同意します。

- 漢城大学校 LMS の利用規約および関連法令に違反しない範囲でのみ本拡張機能を使用すること
- 本拡張機能により収集・表示される情報はユーザー本人の学事情報であり、その正確性の最終確認はユーザー自身の責任で行うこと
- Chrome ウェブストアまたは公式 GitHub リポジトリ以外で配布された非公式版・改変版の使用に伴う責任はユーザー本人が負うこと

## 4. 禁止行為

以下の行為は禁止されます。

- 本拡張機能をリバースエンジニアリングし、漢城大学校 LMS のセキュリティを回避または LMS に不正な影響を及ぼす行為
- 本拡張機能を通じて第三者のアカウントや資料に無断でアクセスする行為
- ライセンス条件に反して、本拡張機能のコードを無断で複製・配布し商業的利益を得る行為

## 5. 免責事項

- 本拡張機能は「**現状有姿(AS IS)**」で提供され、明示・黙示を問わずいかなる保証も行いません。
- 漢城大学校 LMS の構造変更、Google API ポリシーの変更、ユーザー環境の差異等により、機能の一部または全部が正常に動作しないことがあります。
- 本拡張機能の情報表示の誤り、欠落、通知の不達等により発生した出席漏れ・課題未提出等の学事上の不利益について、開発者は一切責任を負いません。

## 6. 責任の制限

法令で認められる最大限の範囲において、開発者は本拡張機能の使用または使用不能から生じる直接的・間接的・偶発的・結果的損害(データ損失、学事上の不利益を含むがこれに限らない)について一切の責任を負いません。

## 7. サービスの変更および中断

開発者は事前の通知なしに本拡張機能の機能を変更、追加、中止することができます。漢城大学校 LMS または Google のポリシー変更によりサービスが中断されることがあります。

本規約は随時更新されることがあり、変更内容は本ページにて掲示されます。変更後も本拡張機能を継続して使用する場合、変更された規約に同意したものとみなされます。
