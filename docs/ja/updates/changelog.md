# アップデートログ

## v5.0.3

- 出席認定要求時間カラムがない講座で出席なのに欠席と表示されるバグを修正
- 出席簿テーブルのカラムを動的に検出し、5列/6列テーブルの両方に対応
- 非トラッキング講座のデータがローカルストレージに蓄積し続ける問題を修正
- Google未ログインユーザーに不要なエラーログが蓄積される問題を修正
- ログダウンロード時にトラッキング講座データのスナップショットを含める
- 未使用の依存関係の削除およびバンドル分割
- アップデート時のdotbugiページ自動表示を削除

## v5.0.2

- 初回インストール/アップデート時に科目が自動トラッキングされないバグを修正
- 新しく追加された科目の自動トラッキング、削除された科目の自動除去
- 過去学期の講義で現在の週として誤って表示されるバグを修正
- 拡張機能アイコンクリック時にユーザー言語設定に基づくドキュメントページへ移動

## v5.0.0

- 多言語対応（韓国語/英語/日本語/中国語）
- 科目別トラッキング選択機能
- クイズ提出トラッキングおよびタスク非表示機能
- 講義ページのステータスバッジおよび現在の週ハイライト
- ポップオーバー設定タブおよびカレンダー連携の改善
- VOD視聴進捗トラッキングの改善
- ネットワークエラー時の自動リトライおよび部分的な失敗の許容
- ログダウンロード機能
- 過去学期の科目の自動フィルタリング
