# HSU 돋부기 이용약관

::: info
[**돋부기**](https://github.com/hs-shell/dotbugi) 이용약관
:::

**시행일:** 2026/05/13

## 1. 개요

본 약관은 사용자가 "HSU 돋부기"(이하 "본 확장 프로그램")를 이용함에 있어 필요한 조건과 책임을 규정합니다. 본 확장 프로그램을 설치·사용함으로써 사용자는 본 약관에 동의한 것으로 간주됩니다.

## 2. 서비스 내용

본 확장 프로그램은 한성대학교 LMS(`https://learn.hansung.ac.kr/`)에서 제공하는 강의, 과제, 퀴즈 정보를 사용자의 브라우저 내에서 통합하여 표시하고, 선택적으로 Google 캘린더와 일정을 동기화하는 기능을 제공합니다.

본 서비스는 **무료**로 제공되며, 한성대학교 또는 Google과 공식적인 제휴 관계가 없는 **비공식 서드파티 도구**입니다.

## 3. 사용자 책임

사용자는 다음 사항에 동의합니다:

- 본 확장 프로그램을 한성대학교 LMS 이용약관 및 관련 법령에 위배되지 않는 범위 내에서만 사용합니다.
- 본 확장 프로그램을 통해 수집·표시되는 정보는 사용자 본인의 학사 정보이며, 정확성에 대한 최종 확인 책임은 사용자에게 있습니다.
- 본 확장 프로그램의 출처(Chrome 웹 스토어 또는 공식 GitHub 저장소) 외에서 배포된 변형본의 사용으로 인한 책임은 사용자 본인에게 있습니다.

## 4. 금지 행위

다음 행위는 금지됩니다:

- 본 확장 프로그램을 역공학(reverse engineering)하여 한성대학교 LMS의 보안 정책을 우회하거나 시스템에 부정한 영향을 끼치는 행위
- 본 확장 프로그램을 통해 타인의 계정·자료에 무단으로 접근하는 행위
- 본 확장 프로그램의 코드를 무단으로 복제·배포하여 상업적 이익을 취하는 행위 (라이선스 조건을 준수하지 않는 경우)

## 5. 면책 조항

- 본 확장 프로그램은 "**있는 그대로(AS IS)**" 제공되며, 명시적이거나 묵시적인 어떠한 보증도 제공하지 않습니다.
- 한성대학교 LMS의 구조 변경, Google API 정책 변경, 사용자 환경 차이 등으로 인해 일부 또는 전체 기능이 정상 작동하지 않을 수 있습니다.
- 본 확장 프로그램의 정보 표시 오류, 누락, 알림 미수신 등으로 인해 발생한 출석 누락·과제 미제출 등 학사상의 불이익에 대해 개발자는 책임을 지지 않습니다.

## 6. 책임의 제한

법령이 허용하는 최대 범위 내에서, 개발자는 본 확장 프로그램의 사용 또는 사용 불가로 인하여 발생한 직접·간접·우발적·결과적 손해(데이터 손실, 학사상 불이익 등을 포함하되 이에 한정되지 않음)에 대해 어떠한 책임도 지지 않습니다.

## 7. 서비스 변경 및 중단

개발자는 사전 통지 없이 본 확장 프로그램의 기능을 변경, 추가, 중단할 수 있습니다. 한성대학교 LMS 또는 Google의 정책 변경으로 인해 서비스가 중단될 수 있습니다.

## 8. 라이선스

본 확장 프로그램의 소스 코드 라이선스는 [GitHub 저장소](https://github.com/hs-shell/dotbugi)의 LICENSE 파일을 따릅니다.

## 9. 준거법 및 분쟁 해결

본 약관은 대한민국 법령에 따라 해석되며, 본 약관과 관련하여 분쟁이 발생한 경우 대한민국 법원을 관할 법원으로 합니다.

## 10. 약관 변경

본 약관은 수시로 업데이트될 수 있으며, 변경 사항은 본 페이지를 통해 게시됩니다. 변경 후 본 확장 프로그램을 계속 사용하는 경우 변경된 약관에 동의한 것으로 간주됩니다.
