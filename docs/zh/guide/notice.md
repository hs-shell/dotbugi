# 公告

## 关于LMS出勤认定系统变更的说明

近期，学校LMS加强了对使用外部倍速程序进行异常观看的监管。

部分用户可能会担心돋부기的自动播放功能是否属于此类情况，特此说明。

### 돋부기不是倍速程序

돋부기的自动播放功能与倍速程序的对比如下：

| 区分 | 外部倍速程序 | 돋부기自动播放 |
|------|------------|--------------|
| 播放速度 | 改为2倍速、4倍速等 | **保持1倍速（原始速度）** |
| 播放时间操纵 | 缩短播放时间 | **无任何操纵** |
| LMS记录 | 与实际学习时间不一致 | **播放时间 = 学习时间 完全一致** |
| 出勤认定 | 有不被认定的风险 | **正常认定** |

### 돋부기会做的事

- 将未观看的课程**按顺序自动连续播放**（点击自动化）
- 已出勤的课程**自动跳过**
- 从之前观看的位置**继续播放**

### 돋부기不会做的事

- 更改视频播放速度
- 操纵播放时间或学习记录
- 绕过或篡改LMS系统

::: tip 总结
돋부기的运作方式与用户手动按下播放按钮**完全相同**。它仅仅是将多个课程连续播放的便利功能，不会以任何方式干预视频速度或学习记录。
:::
