# Google 日历同步

## 连接

1. 打开仪表盘的 **设置** 标签页。
2. 点击"Google Calendar 同步"的 **登录** 按钮。
3. 使用 Google 账号登录后即完成同步连接。

<img src="/images/setting-tab.png" alt="刷新" style="max-width:320px;" />

## 同步

连接后，点击下拉菜单中的 **同步任务** 按钮，待办事项将被添加到 Google 日历中。

<img src="/images/sync.png" alt="同步" style="max-width:320px;" />

同步结果通知：

- "已添加 N 个日程" — 添加了新项目
- "已是最新状态" — 没有需要添加的项目
- "N 个日程添加失败" — 部分失败

## 日历颜色区分

| 颜色      | 项目        |
| --------- | ----------- |
| 💜 薰衣草 | 在线课程 |
| 💙 蓝色   | 作业        |
| 🧡 橙色   | 测验        |

## 登录过期

Google 登录过期时会显示通知。请点击"重新登录"按钮进行重新连接。

<img src="/images/expired.png" alt="过期" style="max-width:320px;" />

## 取消同步

在设置标签页中点击 **登出** 按钮即可取消同步连接。
