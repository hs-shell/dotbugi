# Dotbugi 服务条款

::: info
[**Dotbugi**](https://github.com/hs-shell/dotbugi) 服务条款
:::

**生效日期:** 2026/05/13

## 1. 概述

本条款规定了用户使用"HSU Dotbugi"(以下简称"本扩展程序")时所需的条件与责任。用户安装并使用本扩展程序即视为同意本条款。

## 2. 服务内容

本扩展程序将汉城大学 LMS(`https://learn.hansung.ac.kr/`)提供的课程、作业、测验信息在用户的浏览器内统一展示,并可选与 Google 日历同步日程。

本服务**免费**提供,是与汉城大学或 Google 没有正式合作关系的**非官方第三方工具**。

## 3. 用户责任

用户同意:

- 仅在不违反汉城大学 LMS 使用条款及相关法律法规的范围内使用本扩展程序;
- 通过本扩展程序收集和显示的信息为用户本人的学事信息,准确性的最终确认责任由用户承担;
- 在 Chrome 应用商店或官方 GitHub 仓库以外渠道分发的非官方版本或改版本所造成的责任由用户本人承担。

## 4. 禁止行为

禁止以下行为:

- 通过对本扩展程序进行逆向工程,绕过汉城大学 LMS 的安全策略或对 LMS 造成不当影响;
- 通过本扩展程序未经授权访问他人账户或资料;
- 违反许可证条款,未经授权复制和分发本扩展程序代码以获取商业利益。

## 5. 免责声明

- 本扩展程序按"**现状(AS IS)**"提供,不提供任何明示或默示的担保。
- 由于汉城大学 LMS 结构变更、Google API 政策变更或用户环境差异等原因,部分或全部功能可能无法正常工作。
- 因本扩展程序的信息显示错误、遗漏、通知未送达等导致的出勤缺失、作业未提交等学事方面的不利后果,开发者概不负责。

## 6. 责任限制

在法律允许的最大范围内,开发者对因使用或无法使用本扩展程序而产生的直接、间接、偶发或后果性损害(包括但不限于数据丢失、学事方面的不利后果)概不承担任何责任。

## 7. 服务变更和中止

开发者可在不另行通知的情况下变更、增加或终止本扩展程序的功能。因汉城大学 LMS 或 Google 政策变更,服务可能中断。

## 8. 许可证

本扩展程序的源代码许可证遵循 [GitHub 仓库](https://github.com/hs-shell/dotbugi)的 LICENSE 文件。

## 9. 准据法及争议解决

本条款依据大韩民国法律解释。因本条款产生的争议由大韩民国法院管辖。

本条款可能不时更新,变更内容将通过本页面公布。变更后继续使用本扩展程序即视为同意修改后的条款。
