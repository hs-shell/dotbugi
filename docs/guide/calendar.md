# Google 캘린더 연동

## 연결하기

1. 대시보드의 **설정** 탭을 엽니다.
2. "Google Calendar 연동"의 **로그인** 버튼을 클릭합니다.
3. Google 계정으로 로그인하면 연동이 완료됩니다.

<img src="/images/setting-tab.png" alt="새로고침" style="max-width:320px;" />

## 동기화

연동 후 드롭다운 메뉴의 **태스크 동기화** 버튼을 누르면 할 일이 Google 캘린더에 추가됩니다.

<img src="/images/sync.png" alt="동기화" style="max-width:320px;" />

동기화 결과 알림:

- "N개 일정이 추가되었어요" — 새 항목이 추가됨
- "이미 최신 상태예요" — 추가할 항목 없음
- "N개 일정 추가에 실패했어요" — 일부 실패

## 캘린더 색상 구분

| 색상      | 항목        |
| --------- | ----------- |
| 💜 라벤더 | 온라인 강의 |
| 💙 파랑   | 과제        |
| 🧡 주황   | 퀴즈        |

## 로그인 만료

Google 로그인이 만료되면 알림이 표시됩니다. "다시 로그인" 버튼을 눌러 재연결하세요.

<img src="/images/expired.png" alt="만료" style="max-width:320px;" />

## 연동 해제

설정 탭에서 **로그아웃** 버튼을 클릭하면 연동이 해제됩니다.
