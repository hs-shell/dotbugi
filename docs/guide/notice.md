# 공지사항

## LMS 출석 인정 시스템 변경에 대한 안내

최근 학교 LMS에서 외부 배속 프로그램을 이용한 비정상적인 수강에 대한 단속이 강화되었습니다.

일부 사용자분들이 돋부기의 자동 재생 기능이 이에 해당하는지 우려하실 수 있어 안내드립니다.

### 돋부기는 배속 프로그램이 아닙니다

돋부기의 자동 재생 기능은 다음과 같이 동작합니다:

| 구분 | 외부 배속 프로그램 | 돋부기 자동 재생 |
|------|-------------------|-----------------|
| 재생 속도 | 2배속, 4배속 등으로 변경 | **1배속 (원래 속도) 유지** |
| 재생 시간 조작 | 재생 시간을 단축시킴 | **조작 없음** |
| LMS 기록 | 실제 학습시간과 불일치 | **재생시간 = 학습시간 동일** |
| 출석 인정 | 불인정 위험 | **정상 인정** |

### 돋부기가 하는 것

- 미수강 강의를 **순서대로 자동 연속 재생** (클릭 자동화)
- 이미 출석한 강의는 **자동 스킵**
- 이전에 시청한 위치부터 **이어서 재생**

### 돋부기가 하지 않는 것

- 영상 재생 속도 변경
- 재생 시간 또는 학습 기록 조작
- LMS 시스템 우회 또는 변조

::: tip 요약
돋부기는 사용자가 직접 재생 버튼을 누르는 것과 **완전히 동일하게** 동작합니다. 단지 여러 강의를 연속으로 재생해주는 편의 기능일 뿐, 영상 속도나 학습 기록에는 일절 관여하지 않습니다.
:::
