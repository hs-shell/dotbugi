AI 실수나 교훈을 `.claude/rules/`에 영구 규칙으로 반영하는 스킬.

## 입력

사용자가 다음 중 하나 제공:

- 구체적인 실수 (예: "fetch 모듈에서 키 생성기 안 거치고 dedupe 깨졌어")
- 교훈/규칙 (예: "LMS 파싱은 셀렉터 대신 키워드 기반으로")
- 인자 없이 호출 → 내용을 질문

## 절차

1. **분류**: 어떤 rules 파일에 해당하는지 판단

   - `common.md` — 언어 무관 공통 규칙
   - `typescript.md` — TypeScript/React 관련
   - 해당 파일이 없으면 새로 생성(paths 스코프 포함)

2. **규칙 작성**: 한 줄로 간결하게

   - 형식: `- **금지/필수/권장**: 구체적 규칙 — 이유`
   - 예: `- **필수**: 중복 제거는 makeVodKey/makeAssignKey/makeQuizKey 키 생성기를 거친다 — 키 불일치로 중복 누락 경험`

3. **해당 rules 파일에 추가**: `## 교훈` 섹션이 없으면 파일 끝에 생성

4. **확인**: 추가된 규칙을 사용자에게 보여주고 확인
